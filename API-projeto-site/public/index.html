Summoner's Guide

O que é League of Legends?

League of Legends é um jogo de estratégia em que duas equipes de cinco poderosos Campeões se enfrentam para destruir a base uma da outra. Escolha entre mais de 140 Campeões para realizar jogadas épicas, assegurar abates e destruir torres conforme você luta até a vitória.

Summoner's Rift

Este é o mapa clássico de LoL e a maior referência quando falamos de partidas competitivas no mundo dos eSports, colocando 2 equipes compostas por 5 Campeões cara a cara para uma disputa que se desdobra por múltiplos caminhos, fases e regiões. Após muito suor, o objetivo final será a destruição do Nexus inimigo.
